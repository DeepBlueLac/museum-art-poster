# Atelier Zero 分发与增长

## 核心关键词

`museum poster maker`、`public domain art poster`、`art wallpaper maker`、`printable museum art`、`exhibition poster generator`、`free art poster maker`。

## 首版页面

- 首页：直接完成搜索与海报生成。
- 两个任务指南：展览海报、艺术壁纸；提供独立建议和编辑器预设，不复制首页文案。
- 六个精选作品页：包含作品背景、推荐构图、推荐画幅和直接进入编辑器的链接。

首版不从 API 批量生成六万条页面。只有具备独立排版建议和真实预设的页面进入 sitemap。

## 分享闭环

- URL 只保存作品 ID、模板、画幅、缩放与焦点，不保存原始搜索词。
- 手机优先分享 PNG；不支持文件分享时复制恢复链接。
- 免费海报底部保留克制的 `ATELIER ZERO / ARTIC.EDU` 来源行。

## 衡量

记录 `studio_viewed`、`artwork_selected`、`poster_rendered`、`poster_downloaded`、`poster_shared`、`print_export_interest`。不记录搜索词、IP、精确位置、图片内容或浏览器指纹。
