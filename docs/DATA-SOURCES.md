# 数据与授权

| 来源 | 用途 | 认证 | CORS / 额度 | 授权边界 | 降级 |
|---|---|---|---|---|---|
| [Art Institute of Chicago API](https://api.artic.edu/docs/) | 作品搜索与元数据 | 无 | API 支持浏览器访问；请求防抖并限制为每秒最多一次 | 只接受 `is_public_domain=true` 且有图像的记录 | 使用精选作品目录与当前海报 |
| AIC IIIF Image API | 预览与 Canvas 导出 | 无 | 使用 API 返回的 `config.iiif_url`；优先 843/1686 推荐尺寸 | 只处理上述公共领域作品；保留艺术家、作品、日期和馆藏说明 | 保留上一个成功图像并提示重试 |

不读取 AIC 的长描述作为本站内容，不把未明确公共领域的图片送入导出流程。馆方图像或元数据下架后，精选目录应在下一次发布中移除对应条目。
