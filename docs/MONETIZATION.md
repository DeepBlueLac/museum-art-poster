# 变现与广告边界

## 当前阶段

Atelier Zero 首版处于 `paid-export / hypothesis`。免费提供社交尺寸 PNG，只记录用户对 4K、A4/A3 与无品牌导出的兴趣；不接支付、不展示价格、不承诺打印履约。

## 继续门槛

- 至少 25 次有效作品选择。
- 选择作品后的下载或分享率达到 25%。
- 至少 5 次明确的高分辨率导出兴趣。

满足门槛后，下一步只验证一次性付费导出，不引入账户和订阅。

## 广告边界

首版禁用广告和付费买量。未来若验证广告模式，广告只能出现在用户获得主要价值之后，必须与下载/分享操作有清晰视觉区隔；空状态、错误状态和编辑器主操作附近不得放广告，也不得使用鼓励点击文案。个性化广告启用前必须实现适用地区的同意管理、隐私页、条款页、联系页和有效 `ads.txt`。
