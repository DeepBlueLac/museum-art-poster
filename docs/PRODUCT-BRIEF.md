# Atelier Zero 产品说明

## 一句话产品

Atelier Zero 帮助用户搜索公共领域馆藏名画，并在一分钟内生成可下载、可分享、有清晰来源说明的艺术海报。

## 用户与任务

- 用户：想做艺术壁纸、社交图或家居打印文件，但不会使用专业设计软件的英语用户。
- 输入：作品、艺术家或主题关键词。
- 输出：三种构图、三种画幅中的一张 PNG 海报，以及可恢复编辑状态的分享链接。
- 频率：偶发使用，作品长尾页和分享链接提供回访入口。

## 首版范围

1. 只接 Art Institute of Chicago API 与 IIIF。
2. 仅允许 `is_public_domain=true` 且有 `image_id` 的作品进入编辑器。
3. 提供 Exhibition、Full Bleed、Archive 三种构图。
4. 提供 4:5、1:1、9:16 三种画幅与有限的缩放、焦点调整。
5. 支持 PNG 下载、系统分享、复制链接和少量精选作品/任务页。

## 明确不做

- 不聚合多个博物馆，不上传用户图片。
- 不做自由图层、通用排版编辑器、AI 滤镜或模板市场。
- 不做账号、云同步、社区、打印履约或真实支付。
- 不生成数万条只有标题和图片的薄 SEO 页面。

## 变现假设

免费提供社交尺寸导出；用 `print_export_interest` 验证 4K、A4/A3、无品牌版本的单次付费意愿。任何收入与转化均为待验证假设。

## 14 天验证

- 搜索成功用户中至少 35% 选择一幅作品。
- 选择作品用户中至少 25% 下载或分享。
- 至少 10 次下载/分享和 5 次高分辨率兴趣点击。

## 停止条件

- 14 天内少于 25 次有效作品选择，停止扩数据源。
- 100 名访客少于 5 次下载或分享，重新定位用户任务。
- 作品授权字段出现不确定性时，立即停止对应作品的预览和导出。

## 风险

风险 B：无登录、无上传、无 Key；主要风险来自上游 API 可用性、公共领域标记和跨域 Canvas 导出。
