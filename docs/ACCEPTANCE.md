# Atelier Zero 验收标准

- A-001：服务层拒绝非公共领域或无 `image_id` 的作品。
- A-002：分享链接能恢复作品、模板、画幅、缩放和焦点；非法值安全回退。
- A-003：三种画幅均导出正确尺寸且像素非空的 PNG。
- A-004：1440×900 同屏显示搜索、完整海报和导出操作。
- A-005：390×844 无横向滚动，触控目标至少 44px。
- A-006：长标题、长作者不会越出海报安全区。
- A-007：搜索失败时当前海报和精选作品仍可用。
- A-008：分享/下载反馈不推动工具栏布局。
- A-009：Factory `-Growth` 审计通过。
- A-010：Git 未跟踪环境变量、用户数据或密钥。
- A-011：lint、TypeScript、单测、静态构建通过。
- A-012：发布候选记录提交、回退标签、截图和部署状态。
