# Atelier Zero Design Read

## 产品读法

这是一个工作台，不是名画图库或营销落地页。用户应在 5 秒内看懂：搜索作品、在真实海报上选样式、下载结果。

## 强视觉锚点

海报预览是最高视觉权重。默认态必须展示真实精选作品，搜索、加载或错误都不能让舞台变成空白占位框。

## 品牌记忆

- `ATELIER ZERO` 全大写字标与小型版本号式状态标记。
- 纸白、墨黑、钴蓝、朱红四个固定功能色，艺术品本身提供其余颜色。
- 作品信息像展览图录：清晰、克制、可追溯，不用玻璃拟态和装饰图标圆圈。

## 布局

- 1440×900：288px 搜索栏 + 可伸缩海报舞台 + 304px 样式栏。
- 768×1024：搜索与样式压成顶部双区，海报独占主区域。
- 390×844：品牌、搜索、海报、样式与导出纵向排列；页面无横向滚动。

## 状态

实现 default、hover、focus-visible、active、disabled、loading、empty、success、error、offline fallback。所有反馈占用稳定尺寸，不推动主布局。

## 反模板约束

不复用 P1 的星空、严格灰阶、Poppins、液态玻璃或电影文案；不用紫蓝渐变、三列功能卡、超大营销标题、卡片嵌卡片或全页圆角容器。
